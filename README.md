# Image-Recognition-Decodelabs
Project week 4
Basic Image Recognition using MobileNetV2

This project demonstrates a basic Computer Vision system using the pre-trained MobileNetV2 model.
The system analyzes an input image and predicts the most likely object categories with confidence scores.

The main purpose of this project is to understand how pre-trained deep learning models can perform image classification without training a model from scratch.

Features
Load a pre-trained MobileNetV2 model
Perform image recognition on custom images
Predict top object categories
Display confidence percentages
Visualize input image and prediction results
Technologies Used
Python
TensorFlow
Keras
NumPy
Matplotlib
Pillow
Project Workflow
Load the pre-trained MobileNetV2 model
Load and resize input image
Convert image into array format
Preprocess image for prediction
Perform image recognition
Display top predictions with confidence scores
AI Model Used

This project uses the MobileNetV2 deep learning model trained on the ImageNet dataset.
The model can recognize thousands of real-world objects such as:
Electronics
Animals
Vehicles
Electronics
Everyday objects
